# 🚀 MediConnect Quick Start (HTTPS)

Everything is configured to run on HTTPS for maximum security.

## Start Servers

```bash
./start-https.sh
```

## First Time Setup

1. **Clear browser cache**: http://localhost:3000/clear-cache.html
2. **Login to hospital**: http://localhost:3000/hospital-login
3. **Register patient**: http://localhost:3000/patient-register

## URLs

- **Frontend**: http://localhost:3000
- **Backend API**: https://localhost:8000
- **API Docs**: https://localhost:8000/docs

## Troubleshooting

**401 Errors?** Clear cache: http://localhost:3000/clear-cache.html  
**Certificate Warning?** Click "Advanced" → "Proceed to localhost"  
**Check Status:** `./check-https.sh`
